# 🤖 AI Agent для подбора товаров - ДОКУМЕНТАЦИЯ

## Описание
AI-агент помогает пользователям зоо-магазина подбирать товары, используя интеллектуальный анализ их запросов. Агент отвечает **только** на вопросы о подборе товаров и рекомендациях.

## Особенности

### ✨ Основные функции
- 🎯 Анализирует вопросы пользователей на естественном языке
- 🐾 Определяет вид животного (собаки, кошки, попугаи, рыбы, грызуны)
- 📊 Учитывает возраст питомца (молодой, взрослый, пожилой)
- 🔍 Находит проблему питомца (не ест, нужны витамины, игрушки и т.д.)
- 💰 Фильтрует по бюджету при необходимости
- 🏆 Ранжирует результаты по релевантности

### 🚫 Ограничения
Агент **не отвечает** на вопросы о:
- Контактах и адресах
- Доставке и оплате
- Оформлении заказов
- Гарантиях и возвратах
- Ветеринарных советах (направляет к ветеринару)

## Архитектура

### Файлы системы
```
src/
├── services/
│   └── productRecommender.js      # 🧠 AI-агент для рекомендаций
│
├── data/
│   └── petStore.json              # 💾 База данных товаров для зоо-магазина
│
├── components/ChatBot/
│   ├── chatBotLogic.js            # Обновленная логика с поддержкой AI
│   ├── ChatBot.jsx                # Основной компонент чата
│   ├── ChatMessage.jsx            # Отображение сообщений и рекомендаций
│   ├── ProductRecommendationCard.jsx  # Карточка товара из рекомендации
│   ├── ProductRecommendationCard.module.css
│   └── ChatBot.module.css         # Стили (обновлены)
```

### Поток обработки
```
Пользователь пишет сообщение
    ↓
processUserMessage() (chatBotLogic.js)
    ↓
isProductQuery() проверяет тип запроса
    ├─ Если НЕ о товарах → стандартная обработка
    └─ Если о товарах → getProductRecommendations()
    ↓
AI-агент анализирует запрос:
    ├─ Определяет вид животного
    ├─ Определяет возраст
    ├─ Определяет проблему/потребность
    └─ Получает бюджет (если указан)
    ↓
Фильтрует товары из petStore.json
    ↓
Ранжирует по релевантности
    ↓
Возвращает top-3 рекомендации
    ↓
ChatMessage отображает ProductRecommendationCard
```

## API Функций

### getProductRecommendations(userQuery, options)
**Параметры:**
- `userQuery` (string) - Вопрос пользователя на русском
- `options` (object) - Дополнительные параметры

**Возвращает:**
```javascript
{
  success: boolean,                    // Успешно ли найдены товары
  message: string,                     // Ответ пользователю
  recommendations: Array,              // Массив рекомендованных товаров
  totalFound: number,                  // Всего найдено товаров
  criteria: {                          // Распарсенные критерии
    animalType: string,
    ageGroup: string,
    problems: Array,
    maxPrice: number
  }
}
```

### isProductQuery(query)
**Параметры:**
- `query` (string) - Текст для проверки

**Возвращает:**
- `boolean` - Является ли это запросом о товарах

### getAvailableAnimals()
**Возвращает:**
```javascript
[
  { id: 'dogs', name: 'Собаки' },
  { id: 'cats', name: 'Кошки' },
  ...
]
```

## Модификация логики AI

```javascript
const ANIMAL_KEYWORDS = {
```

## Производительность

- ⚡ Время обработки: < 100ms
- 📊 Поддержка до 100+ товаров
- 🔄 Кэширование данных в памяти
- 📱 Оптимизирован для мобильных устройств

## Известные ограничения

1. Только на русском языке
2. Требует jasne сформулированный запрос
3. Не поддерживает контекст предыдущих сообщений (каждый запрос независим)
4. Максимум 3 рекомендации за раз
