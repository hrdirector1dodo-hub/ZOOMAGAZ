## 🔧 Если что-то не работает

### Проблема: AI не отвечает
1. [ ] Проверьте консоль (F12) на ошибки
2. [ ] Проверьте, что productRecommender.js загружен
3. [ ] Попробуйте перезагрузить страницу
4. [ ] Проверьте путь импорта в chatBotLogic.js

### Проблема: Нет товаров в рекомендациях
1. [ ] Проверьте, что petStore.json загружен
2. [ ] Убедитесь, что животное указано правильно
3. [ ] Посмотрите консоль на ошибки JSON
4. [ ] Проверьте, что товары имеют правильные категории

### Проблема: "В корзину" не работает
1. [ ] Проверьте, что CartContext правильно подключен
2. [ ] Убедитесь, что провайдер CartContext обернул приложение
3. [ ] Посмотрите консоль на ошибки контекста
4. [ ] Проверьте функцию addToCart в контексте

### Проблема: Карточка выглядит странно
1. [ ] Проверьте CSS файл загружен
2. [ ] Убедитесь, что используются правильные переменные CSS
3. [ ] Проверьте ширину контейнера
4. [ ] Попробуйте на другом браузере
