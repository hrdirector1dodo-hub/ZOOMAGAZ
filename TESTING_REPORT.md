### Ошибка #2: CartContext не экспортировался
**Проблема:** 
```javascript
// ❌ Было (не экспортировалось)
const CartContext = createContext(null);

// Использовалось в ChatMessage.jsx:
import { CartContext } from '../../context/CartContext';  // ← ошибка!
```

**Решение:**
```javascript
// ✅ Исправлено (экспортируется)
export const CartContext = createContext(null);
```

**Файл:** [src/context/CartContext.jsx](src/context/CartContext.jsx#L4)  
**Ошибка в браузере:** "The requested module '/src/context/CartContext.jsx' does not provide an export named 'CartContext'"
