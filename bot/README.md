# Telegram-бот EcoPet — Помощник покупателя и аналитик

Этот бот интегрирован с интернет-магазином зоотоваров **EcoPet**. Он использует те же JSON-данные, что и сайт, выступая единым источником информации для клиентов и администратора.

---

## 🚀 Быстрый запуск

### Шаг 1: Получение токена Telegram-бота
1. Откройте Telegram и найдите официального бота [**@BotFather**](https://t.me/BotFather).
2. Отправьте команду `/newbot` и следуйте инструкциям:
   - Введите имя бота (например, `EcoPet Store`).
   - Введите username бота, оканчивающийся на `bot` (например, `ecopet_store_bot`).
3. Скопируйте полученный **Telegram Bot Token** (строка вида `123456789:ABCdefGhIJKlmNoPQRsTUVwxyZ`).

---

### Шаг 2: Настройка окружения
В корневой папке проекта создайте файл с именем `.env` (вы можете скопировать его из `.env.example`) и заполните данные:

```env
TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_ID=ваш_telegram_user_id (можно узнать в @userinfobot)
ADMIN_PASSWORD=секретный_пароль_для_админки (например: admin123)
SITE_URL=http://localhost:5173
```

---

### Шаг 3: Установка зависимостей бота
Для установки библиотек бота перейдите в папку `bot` и запустите установку. Вы можете сделать это одной командой из корня проекта через встроенный портативный Node.js:

```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass; $env:PATH = "node-portable-v22\node-v22.12.0-win-x64;" + $env:PATH; npm --prefix bot install
```

---

### Шаг 4: Запуск сайта и бота

> [!TIP]
> Сайт и бот запускаются в двух разных окнах терминала.

* **Запуск сайта:**
  ```powershell
  Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass; $env:PATH = "node-portable-v22\node-v22.12.0-win-x64;" + $env:PATH; npm run dev
  ```
  *(Сайт будет доступен по адресу: http://localhost:5173)*

* **Запуск бота:**
  ```powershell
  Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass; $env:PATH = "node-portable-v22\node-v22.12.0-win-x64;" + $env:PATH; npm --prefix bot start
  ```

## 🛠️ Функционал и команды бота

### Для клиентов:
- `/start` — Запуск помощника и отображение главного меню (Reply-кнопки).
- `/help` — Описание команд бота.
- `/catalog` / `/categories` — Список категорий с интерактивными кнопками.
- `/search [запрос]` — Поиск по каталогу (например, `/search сухой корм` или просто отправка текста `лежак`).
- `/promotions` — Список действующих акций.
- `/contacts` — Контакты поддержки и ссылки на WhatsApp.
- `/branches` — Адреса, график работы и особенности филиалов.
- `/site` — Быстрая ссылка на интернет-магазин.
- `🛒 Корзина` — Инструкция и ссылка на корзину на сайте.

Каждая карточка товара содержит:
- Фото товара (при наличии).
- Название, бренд, цену, описание и рейтинг.
- Интерактивную кнопку **"Открыть на сайте"** (ведет прямо на карточку товара `/product/{id}`).
- Кнопку **"Похожие товары"** (показывает товары из той же категории).

### Для администратора:
Нажмите кнопку `👤 Админ-панель` или введите команду `/admin`.
- Если ваш Telegram User ID совпадает с `ADMIN_ID` из `.env`, вы будете авторизованы автоматически.
- Если ID не совпадает или не задан, бот попросит ввести секретный `ADMIN_PASSWORD`.

После авторизации доступно меню админа:
1. **📊 Статистика** (`/stats`) — общий объем каталога, средняя цена, топ дорогих/дешевых товаров, количество по категориям, остатки.
2. **📝 Отчет по каталогу** (`/report`) — процент заполненности описаний/фотографий в разрезе категорий, аномалии цен и рекомендации.
3. **🛒 Список товаров** — краткий текстовый дамп товаров.
4. **💡 Рекомендации** — автоматические советы по доработке сайта (закупки товаров с низким остатком, заполнение пустых описаний и исправление ценовых ошибок).
