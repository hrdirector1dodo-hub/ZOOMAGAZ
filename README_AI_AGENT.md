# 🤖 AI-агент для подбора товаров в зоо-магазине

## 🎯 Краткое описание

Создан полнофункциональный **AI-агент для рекомендации товаров**, который:
- 🧠 Анализирует вопросы пользователей на естественном языке
- 🐾 Определяет вид животного, возраст, проблему питомца
- 📊 Подбирает идеальные товары из базы данных
- 💬 Отвечает **только** на вопросы о товарах и рекомендациях
- ✅ Полностью интегрирован в ChatBot

## 💻 Технические детали

### Как это работает
```
Пользователь пишет вопрос
    ↓
isProductQuery() проверяет, о товарах ли это
    ↓
getProductRecommendations() анализирует запрос:
   ├─ Определяет вид животного
   ├─ Определяет возраст питомца
   ├─ Определяет проблему/потребность
   └─ Получает бюджет (если указан)
    ↓
Фильтрует товары из petStore.json
    ↓
Ранжирует по релевантности (scoring)
    ↓
Возвращает Top-3 рекомендации
    ↓
ChatMessage отображает ProductRecommendationCard
    ↓
Пользователь нажимает "В корзину"
```

## 🔒 Безопасность

- ✅ AI отвечает **только** на вопросы о товарах
- ✅ Отклоняет вопросы о контактах, доставке, гарантиях
- ✅ Направляет к ветеринару при упоминании болезней
- ✅ Нет доступа к личным данным пользователя
