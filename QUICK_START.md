# ⚡ БЫСТРЫЙ СТАРТ (2 минуты)

## 🚀 Запуск проекта

### Шаг 1: Установка зависимостей
```bash
npm install
```

### Шаг 2: Запуск сервера
```bash
npm run dev
```

✅ Проект запустится на `http://localhost:5173`

## 📊 Что произойдет

Когда вы отправите вопрос:

1. **AI анализирует** вопрос (< 100ms)
2. **Определяет**: животное, возраст, проблему
3. **Подбирает** подходящие товары
4. **Показывает** 1-3 товара с информацией
5. **Вы нажимаете** "В корзину"

## 🔧 Если что-то не работает

### ❌ Проблема: Чат не открывается
**Решение:**
1. Перезагрузите страницу (F5)
2. Откройте консоль (F12) и посмотрите ошибки
3. Убедитесь, что проект запустился без ошибок

### ❌ Проблема: AI не отвечает
**Решение:**
1. Проверьте консоль (F12)
2. Убедитесь, что файл `productRecommender.js` загружен
3. Попробуйте другой браузер

### ❌ Проблема: Нет рекомендаций
**Решение:**
1. Проверьте, что животное указано правильно
2. Убедитесь, что `petStore.json` загружен
3. Посмотрите консоль на ошибки JSON
