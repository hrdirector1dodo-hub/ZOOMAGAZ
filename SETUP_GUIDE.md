## 📋 Требования

- Node.js v20+ или v22+
- npm или yarn
- Браузер с поддержкой ES6+

## 🔧 Установка и запуск

### Шаг 1: Убедитесь, что зависимости установлены
```bash
npm install
```

### Шаг 2: Запустите проект
```bash
npm run dev
```

Проект должен быть доступен на `http://localhost:5173` (или указанном в консоли порту)

### Шаг 3: Откройте ChatBot
1. Найдите кнопку чата (иконка 🐾 в нижнем правом углу)
2. Нажмите на неё
3. Начните общение с AI-агентом

## 🔍 Как AI работает (Техническое объяснение)

### Шаг 1: Анализ запроса
```javascript
// AI парсит запрос и извлекает:
- Ключевые слова животного (собака, кошка, попугай, рыба, грызун)
```

## 🚨 Возможные проблемы

### Проблема: AI не распознает мой вопрос
**Решение:** Используйте более конкретные слова. Например:
- ❌ "Помоги" → ✅ "Щенку нужен корм"
- ❌ "Товар" → ✅ "Кошке нужны витамины"

### Проблема: Нет товаров в рекомендациях
**Решение:** 
1. Проверьте, что petStore.json загружается без ошибок
2. Убедитесь, что животное указано правильно
3. Откройте консоль (F12) и посмотрите ошибки

### Проблема: Кнопка "В корзину" не работает
**Решение:** 
1. Убедитесь, что CartContext правильно подключен
2. Проверьте, что функция `addToCart` определена в контексте
